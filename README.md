# Uncertainty-Aware Flood Susceptibility Mapping

## Description
This project develops a machine learning framework using Random Forest
to map flood susceptibility and model uncertainty for coastal Karnataka
using Google Earth Engine.

## Technologies Used
- Google Earth Engine (JavaScript API)
- Random Forest
- Sentinel-1 SAR
- SRTM DEM
- ERA5, SMAP

## Study Area
Dakshina Kannada, Udupi, Uttara Kannada

## Output
- Flood Susceptibility Map
- Model Uncertainty Map
- Interactive GEE Web Application
